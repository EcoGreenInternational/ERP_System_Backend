# ERP-System---Backend
Enterprise Resource Planning (ERP) backend service — REST API built with Node.js, Express.js and MongoDB Atlas

## Setup
1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in values
4. Run `npm run dev`

## API Base URL
`http://localhost:5000/api`
